## Run the app
Use command
```bash
git clone https://github.com/concerned-guy/USTH-SE-2018
cd USTH-SE-2018/Team-1337
npm install
npm start
```
And then, in your favorite browser, visit the following URL

[http://localhost:3000/](http://localhost:3000/)
